# カリフォルニア海外研修ウェブサイト

## 別のパソコンでクローンする方法

### 1. コマンドでクローン

デスクトップや任意のフォルダで以下のコマンドを実行：

```powershell
git clone https://github.com/kawatarui/uiux-america-web.git
```

### 2. VS Codeでクローン

1. VS Codeを開く
2. `Ctrl + Shift + P` を押す
3. 「Git: Clone」と入力して選択
4. リポジトリURL を入力: `https://github.com/kawatarui/uiux-america-web.git`
5. 保存先フォルダを選択
6. 「Open」をクリック

### 3. GitHubからZIPでダウンロード

1. https://github.com/kawatarui/uiux-america-web にアクセス
2. 緑色の「Code」ボタンをクリック
3. 「Download ZIP」を選択
4. ZIPを解凍

## ファイルを開く

```powershell
cd uiux-america-web
Start-Process Untitled-1.html
```

## 変更を保存

```powershell
git add .
git commit -m "変更内容"
git push
```
